# PR naming convention
Follow that keyword with a "/" then the issue number, then and "_" then use kabob case for the rest of the branch name.

Resolves #issue_number
**Examples**: 
- Feat/#3_add-readme
- Perf/#40_fix-slow-component
- Fix/#22_padding-error
